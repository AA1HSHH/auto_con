# NJUPT自动连接脚本
## 环境安装
```
conda create -n net python=3.8
conda activate net
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple psutil requests 
```
## 执行
第一个参数是conda环境的python位置，第二个参数是文件位置
